# Nuxt.js

::: warning
The Nuxt module and this docs are a WIP. Things might not work yet.
:::

Install

```bash
npm install nuxt-vuefire
```

Add it to your nuxt config:

```ts
export default defineNuxtConfig({
  modules: [
    // ... other modules
    'nuxt-vuefire',
  ],

  vuefire: {
    /* options */
  },
})
```
